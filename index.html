Welcome to the Platipy Project

The Platipy Project is an effort to bring high quality libraries and documentation for game development to the OLPC XO platform. This page is currently under heavy construction. For now, the following temporary pages may provide useful information.
  • Platipy Documentation - The documentation for many of the tools and libraries provided by the Platipy Project
  • Platipy's Github - Source code for all the libraries, documentation, and publications related to Platipy
  • Platipy's OLPC Contributor Application - Summarizes briefly some goals of the Platipy project.
  • UD Games - A collection of games written by students in an educational game development course at the University of Delaware. Platipy was used as a part of the development of the games in recent semesters of this course.
